## **说明**

|     类型      |                  说明                  |
| :-----------: | :------------------------------------: |
|  `function`   |                  函数                  |
| `function_T`  |                模板函数                |
|    `class`    |                 类类型                 |
|   `class_A`   |                 抽象类                 |
|   `class_T`   |                 模板类                 |
|  `delegate`   |            代理（函数指针）            |
|    `enum`     |                  枚举                  |
|  `interface`  |                  接口                  |
| `interface_T` |                模板接口                |
|   `struct`    |                 结构体                 |
|  `struct_T`   |               模板结构体               |
|    `value`    | 值类型（基类型的别名，不产生新的类型） |
|               |                                        |
