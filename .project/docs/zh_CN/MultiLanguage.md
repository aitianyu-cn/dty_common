## 静态多语言

```
    文件目录：${项目根目录}/native/res/i18n/*
    文件要求：标准的json格式文件
    格    式：
        {
            "多语言文本键": {
                "语言类型": "该语言对应的替换文本",
                "default": "默认状态下的替换文本"
            }
        }
```

```

    注意：
    多语言文本键 ：该值必须为唯一字符串，并符合C++语言宏定义要求，可支持宏函数替换
    语言类型     ：当前只支持中文（ch）和英文（en）
    default     ：该值指示了在对应语言类型未能匹配的情况下默认的文本
```
